void acc_init ( uint8_t  __cs_pin,
ACC_POWER_MODE  __power_mode,
ACC_OUTPUT_DATA_RATE  __output_data_rate,
ACC_FULL_SCALE  __full_scale 
)

Initializes the driver.

Sets up the SPI and writes the setup parameters to the accelerometer chip Sending configuration data to the accelerometer using multi-write. index 0 : address and multiple byte data bit index 1-5 : data for the different control registers

: Further information can be found on pages 24-28 in the datasheet

Parameters
uint8_tcs_pin - chip select pin
ACC_POWER_MODEpower_mode - accelerometer power mode
ACC_OUTPUT_DATA_RATEoutput_data_rate - the rate the data is updated
ACC_FULL_SCALEfull_scale - the full scale g measurement
