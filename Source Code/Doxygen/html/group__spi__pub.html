Sets up the SPI bus as master using supplied parameters.

Takes the supplied SPI parameters and stores them in a handle_param array and returns a handle

Note
Max 16 different handles are available
Parameters
SPI_DATA_MODE__mode - SPI mode
SPI_DATA_DIRECTION__data_direction - the data direction (MSB first or LSB first)
SPI_DIVIDER__freq_devider - the frequency divider
uint8_t__cs_pin - the pin number of the CS/CE pin
SPI_CS_ACTIVE_LEVEL__cs_active_level - the level of which the chip is active (high/low)
void*__callback_function_ptr - the pointer to the function which should be called when the SPI interrupt is triggered
Returns
uint8_t - a handle for the setup
