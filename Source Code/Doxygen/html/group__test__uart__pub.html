Starts the testing routine for uart0 and uart1.

Sets up both uart0 and uart1, sends data on round trip through both uarts and checks that the result is the same as the __uart_compare_string

Parameters
constchar *__uart_test_string - string to be send
constchar *__uart_compare_string - string to be compared to
Returns
bool - true = passed; false = failed
Note
To run this test uart0 and uart1 must be connect crossed. TX0->RX1, TX1->RX0
